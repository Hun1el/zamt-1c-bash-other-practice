## 📋 Объединенная учебная практика ZAMT

Этот репозиторий содержит **несколько учебных практик**, связанных с разработкой, автоматизацией и работой с серверами.

Практика включает работу с различными инструментами и технологиями:

* разработка на **1C:Enterprise**
* написание **Bash и PowerShell скриптов**
* **автоматизация тестирования**
* основы **серверного администрирования**

---

## 📁 Структура репозитория

```
1C/               ← разработки на платформе 1C:Enterprise
Bash_Server/      ← системные Bash (PowerShell)-скрипты и работа с серварами 
Prod_practice/    ← производственные практики
Testing/          ← автоматизация тестирования
```
